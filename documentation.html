Website Documentation

Welcome to the documentation page of our vegan/vegetarian restaurant website. Here you will find information about the features, structure, and technologies used in this website.

Features

  • Home Page: Introduction and overview of the restaurant.
  • About Us: Information about the restaurant's story and team.
  • Menu: Detailed menu with prices and descriptions of dishes.
  • Sustainability: Information about the restaurant's sustainable practices.
  • Contact: Contact form for reservations and inquiries.

Technologies Used

  • HTML5: Markup language for structuring the content.
  • CSS3: Styling language for designing the website.
  • JavaScript: Programming language for adding interactivity (if applicable).

How to Navigate

Use the navigation bar at the top of each page to quickly access different sections of the website. Each link takes you to a specific page with relevant information.
